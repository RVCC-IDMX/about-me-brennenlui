CSS Animation

CSS Animation allows you to animate HTML elements on your website without using JavaScript. CSS Animation is supported on Chrome (version 43.0 or later), Edge (version 10.0 or later), Mozilla (version 16.0 or later), Safari (version 9.0 or later), and much more. CSS Animation lets gradually changes the CSS styling for element from one style to another. In order to achieve this you will need to use the @keyframes rule. The keyframe will determine how the animated element is render at each given time during the animation. Whatever CSS you put inside of the keyframe will determine how the element will change styles. For example the following code: @keyframes example { from{background-color: white;} to{background-color: black;} } Will make the background of an element turn from white to black.

The animation property lets you configure the animationâ€™s timing, duration, direction, and much more. The animation property does not physically change the appearance of the elements, instead it tells the keyframe what sequence the animation should take place. The animation property has many different sub-properties which are:

  • animation-name
    • Specifies the name for the keyframe
    • animation-name: example; names the keyframe as example
  • animation-delay
    • Specifies how long an animation is delayed before it starts
    • animation-delay: 4s; means that the animation will wait 4 seconds before it begins.
  • animation-iteration-count
    • Specifies how many times an animation should run
    • animation-iteration-count: 5; repeats the animation five times before the animation stops
    • animation-iteration-count: infinite; will repeat an infinite amount of times
  • animation-direction
    • Specifies if an animation should start forwards, backwards, or in alternate cycles
    • animation-direction: normal; will play the animation forwards
    • animation-direction: reverse; will play the animation backwards
    • animation-direction: alternate; will play the animation forwards and then backwards
    • animation-direction: alternate-reverse; will play the animation backwards and then forwards
  • animation-timing-function
    • Specifies the speed curve of the animation
    • animation-timing-function: ease; the animation will start slow, speed up, and the end slowly
    • animation-timing-function: linear; the animation will play at the same speed from start to finish
    • animation-timing-function: ease-in; the animation will start slow
    • animation-timing-function: ease-out; the animation will end slow
    • animation-timing-function: ease-in-out; the animation will start slow and end slow
    • animation-timing-function: cubic-bezir(n,n,n,n) lets you create your own values using a cubic-bezir function
